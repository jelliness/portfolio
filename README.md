# Portfolio

A personal portfolio website showcasing projects, skills, experience, and blog posts. Built with HTML, CSS, and JavaScript.

## Features
- Responsive design
- Project and skills pages
- Contact form
- Animated UI elements

## Structure
- `index.html`: Home page
- `pages/`: About, blog, experience, projects, skills
- `assets/`: CSS and JS files

## Usage
Open `index.html` in your browser to view the site.
